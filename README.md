# Thikaidar Bill Generator

A voice-enabled bill generator for tile work businesses. Create professional bills using voice or text input.

## Features

- 🎙️ **Voice Input** - Speak your answers in Hindi or English
- 📝 **Text Input** - Type answers as an alternative
- 🗣️ **Voice Output** - App reads questions aloud
- 🏠 **Multi-floor/room support** - Add multiple floors and rooms
- 📊 **Live Preview** - See bill update in real-time
- 💾 **Auto-save** - Bills saved to local storage
- 📄 **PDF Export** - Download professional PDF bills
- 📱 **WhatsApp Share** - Share bills directly via WhatsApp (free)
- 🌐 **Bilingual** - Supports Hindi and English

## Getting Started

1. Open `index.html` in a web browser
2. Select your language (Hindi/English)
3. Start a new bill and enter customer details
4. Add floors and rooms with dimensions
5. Enter tile rates
6. Export as PDF or share via WhatsApp

## Optional: Groq API Key

For better voice understanding, you can add a free Groq API key:
1. Get a free key at https://console.groq.com
2. Enter it in the API key field on the welcome screen

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- jsPDF for PDF generation
- Web Speech API for voice input/output
